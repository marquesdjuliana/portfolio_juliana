Projetos

Confira alguns dos projetos que desenvolvi:

  • Aplicação Full Stack de Gerenciamento de Clientes

    Esta aplicação permite aos usuários visualizar, criar e editar clientes. O frontend é construído em React, enquanto o backend utiliza Node.js e MySQL. O projeto inclui validação de dados, gerenciamento de usuários e armazenamento persistente.

  • Restaurant Orders

    Desenvolvido durante o módulo de Ciência da Computação na Trybe, este projeto finalizou um sistema para gerenciamento de pedidos de restaurante. Ele inclui funcionalidades de CRUD (Create, Read, Update, Delete) e utiliza uma arquitetura de microserviços.

  • Trybe Hotel

    Uma aplicação ASP.NET Core desenvolvida para gerenciar reservas de hotéis. O projeto integra com uma API externa para busca de disponibilidade e gerencia reservas, check-ins e check-outs.
